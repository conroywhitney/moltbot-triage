## Description

## Issue

Jarred Sumner identified a memory leak pattern affecting long Claude sessions:
https://x.com/jarredsumner/status/2017825694731145388

**Before (leaks):** `() => controller.abort()`
**After (fixed):** `controller.abort.bind(controller)`

This provides ~1GB memory reduction on long sessions.

## Affected Files

Found ~15 instances in the codebase:
- `infra/fetch.js`
- `infra/provider-usage.fetch.shared.js`
- `agents/tools/web-shared.js`
- `agents/pi-tools.abort.js`

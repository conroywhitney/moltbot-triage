## Description

  After running clawdbot models set anthropic/claude-sonnet-4-5, an update or config normalization changed the model entry to:                                                                
                                                                                                                                                                                              
  "model": {                                                                                                                                                                                  
    "primary": "anthropic/claude-sonnet-4-5@anthropic:claude-cli"                                                                                                                             
  }                                                                                                                                                                                           
                                                                                                                                                                                              
  The agent then fails with:                                                                                                                                                                  
  ⚠️ Agent failed before reply: Unknown model: anthropic/claude-sonnet-4-5@anthropic:claude-cli.                                                                                              
                                                                                                                                                                                              
  Expected: Either don't auto-add the @profile suffix, or parse it correctly in the agent.                                                                                                    
                                                                                                                                                                                              
  Workaround: clawdbot models set anthropic/claude-sonnet-4-5                                                                                                                                 
                                                                                                                                                                                              
  Version: 2026.1.24-3 (beta)

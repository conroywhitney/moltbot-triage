## Description

## Summary

The openclaw-gateway nix package is missing the docs/reference/templates directory, causing the gateway to fail when handling messages.

## Steps to reproduce

1. Install openclaw-gateway via nix-openclaw Home Manager module
2. Run openclaw gateway
3. Send a message to the Telegram bot

## Expected behavior

The gateway should process messages and respond normally.

## Actual behavior

Error: Missing workspace template: AGENTS.md (/nix/store/hacpxjny15dbjn6y9ln5xmijcnf1y6l4-openclaw-gateway-2026.1.8-2/lib/openclaw/docs/reference/templates/AGENTS.md). Ensure docs/reference/templates are packaged.

## Environment

- Clawdbot version: 2026.1.30 (gateway 2026.1.8-2)
- OS: NixOS, x86_64-linux
- Install method: nix-openclaw Home Manager module

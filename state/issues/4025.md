## Description

## Problem
www.cloudwise.ai is an aggregated API platform that integrates multiple large language models, such as the Claude series and the GPT series.
<img width="1280" height="728" alt="Image" src="https://github.com/user-attachments/assets/051940a6-f8e1-44db-a6cd-35dac332aa7c " />

## Solution
How do you call Moltbot (Clawdbot)?
We provide support for OpenAI-completions and Claude-series models.

## Reference Configuration
### **Claude** 
```
{
...
  "auth": {
    "profiles": {
      "anthropic:default": {
        "provider": "anthropic",
        "mode": "api_key"
      }
    }
  },
  "models": {
    "providers": {
      "anthropic": {
        "baseUrl": "https://api.cloudwise.ai/api",
        "apiKey": "ak_****",  // API Key
        "api": "anthropic-messages",
        "models": [
          {
            "id": "claude-opus-4-5-20251101",
            "name": "claude-opus-4-5-20251101"

        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {

          "alias": "anthropic"

      "subagents": {
        "maxConcurrent": 8
```

## Description

## Bug Report

**Summary:** Release v2026.1.24 only contains source code — no macOS app binaries (DMG/ZIP). This breaks the Sparkle auto-update flow for users on v2026.1.23.

**Current state:**
- v2026.1.23: Has `Clawdbot-2026.1.23.dmg` and `.zip` ✅
- v2026.1.24: Source code only, no app assets ❌

**Impact:**
- Sparkle detects v2026.1.24 as available
- Update fails because there's nothing to download
- Users stuck on .23 with a perpetual "update available" state

**Expected:** Each release that bumps version should include macOS app binaries, or Sparkle should not advertise source-only releases.

**Workaround:** Manually stay on v2026.1.23 and ignore update prompts.

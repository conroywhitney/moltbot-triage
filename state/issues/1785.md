# SkillKit Integration Proposal for Clawdbot

## Summary

Proposal to integrate [SkillKit](https://github.com/rohitg00/skillkit) as an optional skill management layer for Clawdbot, enabling cross-agent skill portability and unified skill management across 17 AI coding agents.

**Current Version**: v1.7.2 | **Marketplace**: 15,000+ skills | **Website**: [agenstskills.com](https://agenstskills.com)

## The Problem

Clawdbot users who also use other AI coding agents (Claude Code, Cursor, Codex, Windsurf, etc.) face:

1. **Skill fragmentation** - Skills written for Clawdbot don't work in other agents
2. **Manual duplication** - Users rewrite the same skills for each agent
3. **Format differences** - Each agent has different skill formats and directories
4. **No unified discovery** - Finding and installing skills requires different workflows per agent

## The Solution: SkillKit

SkillKit is a universal CLI that:

- **Translates skills** between 17 agent formats automatically
- **Unifies skill management** - one command installs to all agents
- **Smart recommendations** - analyzes projects and suggests relevant skills
- **Team collaboration** - share skill bundles across teams
- **Session memory** - persistent learnings across agent sessions
- **Marketplace access** - 15,000+ curated skills

### Already Supported

SkillKit already supports Clawdbot as one of its 17 agents:

| Agent | Format | Skills Directory |
|-------|--------|------------------|
| **Clawdbot** | SKILL.md | `skills/` or `~/.clawdbot/skills/` |

## Proposed Integration

### Phase 1: Documentation & Awareness ✅

1. Add SkillKit to Clawdbot docs as a recommended tool for multi-agent users
2. Document how SkillKit complements ClawdHub
3. Add examples of translating skills from other agents to Clawdbot format

### Phase 2: Native Plugin (In PR #1788)

Create an official `skillkit` plugin for Clawdbot that exposes 9 tools:

```
extensions/skillkit/
├── clawdbot.plugin.json
├── index.ts
├── package.json
└── src/tools.ts
```

**Registered Tools:**

| Tool | Description |
|------|-------------|
| `skillkit_search` | Search SkillKit marketplace for skills |
| `skillkit_install` | Install skills with auto-translation to Clawdbot format |
| `skillkit_translate` | Convert skills between 17 agent formats |
| `skillkit_sync` | Sync installed skills to agent config |
| `skillkit_recommend` | Get project-aware skill recommendations |
| `skillkit_list` | List all installed skills |
| `skillkit_context` | Analyze project context for smart recommendations |
| `skillkit_publish` | Publish skills to SkillKit marketplace |
| `skillkit_memory` | Access persistent session memory and learnings |

### Phase 3: Deep Integration (Future)

- Hook into Clawdbot's skill loading pipeline
- Bidirectional sync with ClawdHub
- Multi-agent orchestration for Clawdbot's multi-agent system

| Skill directories | workspace + managed | configurable | ✅ Yes |
| Gating/filters | requires.bins/env/config | metadata | ✅ Adaptable |

## Example Workflow

```bash
# User has skills from Claude Code, wants to use with Clawdbot

# 1. Translate existing skills
skillkit translate ~/.claude/skills/* --to clawdbot

# 2. Or install from marketplace directly
skillkit install anthropics/skills --agent clawdbot

# 3. Skills appear in ~/.clawdbot/skills/
clawdbot skills list  # Shows newly installed skills

# 4. Get project-aware recommendations
skillkit recommend

# 5. Publish your own skills
skillkit publish ./my-skill/SKILL.md
```

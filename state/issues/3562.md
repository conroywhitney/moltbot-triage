## Description

## Summary

Telegram Bot API supports a `disable_notification` parameter that controls 
whether a message triggers a notification sound on the recipient's device. 
Currently Moltbot's message tool doesn't expose this parameter, making it 
impossible to ensure time-sensitive messages (like reminders) trigger an 
audible alert.

## Proposed solution

Add a flag to `moltbot message send` for Telegram channel:

```bash
moltbot message send --channel telegram --target 123456 \
  --message "ðŸš¨ Reminder!" \
  --notify        # or --loud / --alert

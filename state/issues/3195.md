## Description

## Problem

When using Venice AI as a provider, users receive **no feedback** about:
1. Their DIEM balance running low
2. Approaching or hitting API key spending caps  
3. When they've run out of funds entirely

**User feedback:** "There's no UI feedback in the Clawdbot API when an API key runs out of funds or hits its spending cap. I've spent $250 so far today via ClawdBot."

## Venice API Headers (Already Available!)

Venice returns balance info on **every authenticated request**:

```
x-venice-balance-diem: 44.32116826
x-venice-balance-usd: (if applicable)
x-venice-balance-vcu: (if applicable)
x-ratelimit-remaining-requests: 499
x-ratelimit-remaining-tokens: 5000000
```

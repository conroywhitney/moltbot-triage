### @Luoxiaopan (2026-02-01)

This could be due to missing git installation. Install git first (if you are using windows):

> winget install --id Git.Git -e --source winget
> winget install --id GitHub.cli

restart terminal window.

> gh auth login

Then restart the terminal window again and retry install.

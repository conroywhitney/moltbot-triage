## Description

## 概要

Gemini CLI の Conductor 拡張で採用されているコンテキスト駆動開発のワークフローを、OpenClaw のスキルやワークスペース運用に取り入れることを検討する。

## Conductor とは

Google がリリースした Gemini CLI 拡張で、以下の特徴を持つ：

- **コンテキストの永続化**: プロジェクト情報・技術スタック・ワークフロールールを Markdown で管理
- **Track ベースの作業管理**: spec.md / plan.md で仕様と計画を明示化
- **ワークフロー**: Context → Spec/Plan → Implement
- **Git 管理**: 全てバージョン管理・レビュー可能

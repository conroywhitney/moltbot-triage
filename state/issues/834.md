## ğŸš¨ Don't Panic: You haven't been hacked.

First, take a deep breath. Your friends **do not** have access to your server or WhatsApp.

The "pairing code" the bot sent them is just a **connection request**. It is useless unless **YOU** (the owner) manually type that code into your own computer terminal to approve it. Unless you type that code, they can't do anything.

---

## ğŸ› ï¸ How to Fix It (Stop the Spam)

The bot is doing this because, by default, it is in `Open/Pairing Mode`. It thinks every new message is a potential owner trying to connect, so it sends a code to everyone.

**To stop the mass messaging immediately:**

1. Open your config file: `~/.clawdbot/clawdbot.json`
2. Find the `dmPolicy` line and change it from `pairing` to `allowlist`.

```json
// Change this line:
"dmPolicy": "pairing"

// To this:
"dmPolicy": "allowlist"
```

3. **Restart the bot.**

### What does this change do?
Now, the bot will check if the phone number is in your `allowed list` first.

* âŒ **Friend/Stranger:** Bot stays silent. (No spam, no codes).
* âœ… **You (Allowed Number):** Bot replies normally.

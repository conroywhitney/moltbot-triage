## Fix

As suggested [here](https://github.com/hoppscotch/hoppscotch/issues/4960#issuecomment-2780903150) it can be solved by the following patch:

```patch
--- Dockerfile
+++ Dockerfile
@@ -21,7 +21,7 @@ COPY ui/package.json ./ui/package.json
 COPY patches ./patches
 COPY scripts ./scripts

-RUN pnpm install --frozen-lockfile
+RUN pnpm install --frozen-lockfile --package-import-method=hardlink 

 COPY . .
 RUN OPENCLAW_A2UI_SKIP_MISSING=1 pnpm build
```

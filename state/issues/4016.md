## Description

## Summary
As per the title. When I text telegram, I get raw JSON responses instead of normal text messages.

## Steps to reproduce
1. Install clawdbot with Telegram channel configured
2. Send message
3. Receive something like this:
```
{
"name": "tts",
"arguments": {
"text": "Hello!"
}
}
```

## Expected behavior
I expect a text response like "Hello!", in this case

## Environment
- Clawdbot version: 2026.1.24-3
- OS: Ubuntu 24.04.3 LTS (in a VM under Proxmox)
- Install method (pnpm/npx/docker/etc): CLI script
- Using qwen2.5-coder:14b on Ollama running on a local machine (separate)

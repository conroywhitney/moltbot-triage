I'm having the same issue. I managed to find a hack yesterday which was to create a new profile and start the browser, that works.

```
./openclaw.json

  "browser": {
    "enabled": true,
    "executablePath": "/usr/bin/google-chrome-stable",
    "headless": true,
    "noSandbox": true,
    "profiles": {
      "openclaw": {
        "cdpPort": 18800,
        "color": "#FF4500"
      },
      "chrome": {
        "cdpPort": 18800,
        "color": "#00AA00"
      }
    }
```

## Description

Add Soniox as a new audio transcription provider for the media-understanding system. Soniox supports 60+ languages with high accuracy and features like code-switching and speaker diarization.                                                                                                                          
                                                                                                                                                                   
Motivation                                                                                                                                                       
                                                                                                                                                                   
  - https://soniox.com/benchmarks show 6.5% WER in English vs 9.3% for Deepgram                                                                                    
  - Strong multilingual support (60+ languages) with better accuracy than Whisper for non-English                                                                  
  - Code-switching support (mixing languages within sentences)                                                                                                     
  - Speaker diarization and real-time translation capabilities                                                                                                     
                                                                                                                                                                   
Implementation Details                                                                                                                                           
                                                                                                                                                                   
API Workflow                                                                                                                                                     
                                                                                                                                                                   
Unlike OpenAI/Deepgram (synchronous), Soniox uses an async workflow:                                                                                                                                                                                                                                                                
  1. POST /v1/transcriptions → Create job, get transcription_id                                                                                                    
  2. GET /v1/transcriptions/{id} → Poll until status="completed"                                                                                                   
  3. GET /v1/transcriptions/{id}/transcript → Get text                                                                                                             
                                                                                                                                                                   
Base URL: https://api.soniox.com/v1                                                                                                                              
Auth: Authorization: Bearer <api_key>                                                                                                                            
OpenAPI Spec: https://api.soniox.com/v1/openapi.json                                                                                                             
                                                                                                                                                                   
Files to Create/Modify                                                                                                                                           
                                                                                                                                                                   
  1. New: src/media-understanding/providers/soniox/audio.ts - Transcription logic with polling                                                                     
  2. New: src/media-understanding/providers/soniox/index.ts - Provider registration                                                                                
  3. Modify: src/media-understanding/providers/index.ts - Add to registry                                                                                          
  4. Modify: src/media-understanding/runner.ts - Add to AUTO_AUDIO_KEY_PROVIDERS                                                                                   
  5. Modify: src/media-understanding/defaults.ts - Add default model                                                                                               
                                                                                                                                                                   
Request Format                                                                                                                                                   
                                                                                                                                                                   
  {                                                                                                                                                                
    "model": "stt-async-preview",                                                                                                                                  
    "file_id": "...",                                                                                                                                              
    "language_hints": ["en"],                                                                                                                                      
    "enable_speaker_diarization": false                                                                                                                            
  }                                                                                                                                                                
                                                                                                                                                                   
Configuration Example                                                                                                                                            
                                                                                                                                                                   
  tools:                                                                                                                                                           
    media:                                                                                                                                                         
      audio:                                                                                                                                                       
        models:                                                                                                                                                    
          - type: provider                                                                                                                                         
            provider: soniox                                                                                                                                       
            language: sl  # or any of 60+ supported languages

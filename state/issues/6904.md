## Description

## Summary

Channel can not be deleted, can only be disabled


## Steps to reproduce

I run clawdbot at centos
1.run "clawdbot channels remove --channel imessage", the channel's status changes to disabled.
2.run "clawdbot channels remove --channel imessage --delete", it does not work.

## Expected behavior

run "clawdbot channels list"
the channel should not be listed.

## Actual behavior

run "clawdbot channels list"
the channel listed,

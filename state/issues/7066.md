## Description

## Summary
Telegram Bot API supports `<blockquote expandable>` for collapsible quotes since [Bot API 7.3](https://core.telegram.org/bots/api#blockquote), but Clawdbot's markdown formatter escapes HTML tags instead of passing them through.

## Current behavior
When sending a message containing `<blockquote expandable>...\</blockquote>`, the HTML is escaped and rendered as plain text:
```
<blockquote expandable>test</blockquote>
```

## Expected behavior
The `<blockquote expandable>` tag should be rendered as a collapsible quote in Telegram.

## Use case
Useful for morning briefs and long-form notifications where sections can be collapsed to reduce visual clutter.

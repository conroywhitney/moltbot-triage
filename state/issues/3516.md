## Description

  ## Summary                                                                                                                                                                                                  
                                                                                                                                                                                                              
  Adds full support for Zoom Team Chat as a core channel plugin, enabling Moltbot to receive and respond to direct messages via Zoom's Team Chat Bot API. 

Closed PR here: https://github.com/moltbot/moltbot/pull/3279

**Contributor Background**: Implemented by [@pranavk95](https://www.linkedin.com/in/pranavk95/), a Zoom Senior Software engineer with expertise in the Zoom Team Chat platform and bot APIs.  

Demo Video: https://zoom.us/clips/share/j-ffAp2VSMiOcvZiYEFSvw                                                 
                                                                                                                                                                                                              
  ## What's New                                                                                                                                                                                               
                                                                                                                                                                                                              
  ### Core Integration                                                                                                                                                                                        
  - **Core Channel Registration**: Registered Zoom as a core channel alongside Slack, Telegram, Discord, etc.                                                                                                 
  - **Channel Dock**: Added Zoom to the channel dock with proper capabilities and configuration resolvers                                                                                                     
  - **Normalize Helpers**: Implemented `normalizeZoomMessagingTarget()` and `looksLikeZoomTargetId()` following core channel patterns                                                                         
  - **Plugin SDK Exports**: Exported all Zoom types, configs, and helpers for extension consumption                                                                                                           
                                                                                                                                                                                                              
  ### Plugin Implementation                                                                                                                                                                                   
  - **Webhook Server**: Express-based webhook server on port 3001 for receiving `bot_notification` events                                                                                                     
  - **OAuth Flow**: Complete OAuth 2.0 authorization flow with deep linking back to Zoom app                                                                                                                  
  - **Message Handling**: Full integration with `dispatchInboundMessage()` for AI routing and session management                                                                                              
  - **API Client**: Bot token management with caching and automatic refresh                                                                                                                                   
                                                                                                                                                                                                              
  ### Configuration                                                                                                                                                                                           
  - **Onboarding Wizard**: Full QuickStart integration with step-by-step credential collection                                                                                                                
  - **OAuth Redirect URI**: User-configurable `redirectUri` for flexible deployment (ngrok, production domains)                                                                                               
  - **Environment Detection**: Auto-detects dev vs production based on Bot JID (@xmppdev vs @xmpp)                                                                                                            
  - **DM Policies**: Supports `open`, `allowlist`, and `pairing` policies                                                                                                                                     
                                                                                                                                                                                                              
  ### Documentation                                                                                                                                                                                           
  - **Setup Guide**: Comprehensive docs at `/docs/channels/zoom.md` covering:                                                                                                                                 
    - Zoom App Marketplace setup (General App creation)                                                                                                                                                       
    - OAuth and webhook URL configuration                                                                                                                                                                     
    - Local development with ngrok                                                                                                                                                                            
    - Production deployment                                                                                                                                                                                   
    - Troubleshooting common issues                                                                                                                                                                           
  - **Architecture Notes**: Detailed explanation of webhook flow, session management, and security

  ### Key Features                                                                                                                                                                                            
  - ✅ Direct message support                                                                                                                                                                                 
  - ✅ Session management (per-user sessions)                                                                                                                                                                 
  - ✅ Streaming responses (coalesced with 1500 char minimum)                                                                                                                                                 
  - ✅ Webhook verification via secret token                                                                                                                                                                  
  - ✅ OAuth 2.0 app installation flow

  ## Configuration Example                                                                                                                                                                                    
                                                                                                                                                                                                              
  ```json5                                                                                                                                                                                                    
  {                                                                                                                                                                                                           
    channels: {                                                                                                                                                                                               
      zoom: {                                                                                                                                                                                                 
        enabled: true,                                                                                                                                                                                        
        clientId: "YOUR_CLIENT_ID",                                                                                                                                                                           
        clientSecret: "YOUR_CLIENT_SECRET",                                                                                                                                                                   
        botJid: "bot@xmppdev.zoom.us",                                                                                                                                                                        
        secretToken: "YOUR_SECRET_TOKEN",                                                                                                                                                                     
        redirectUri: "https://yourdomain.com/api/zoomapp/auth",                                                                                                                                               
        apiHost: "https://zoom.us",                                                                                                                                                                        
        oauthHost: "https://api.zoom.us",                                                                                                                                                                      
        dm: { policy: "open" }                                                                                                                                                                                
      }                                                                                                                                                                                                       
    }                                                                                                                                                                                                         
  }

## Description

## Summary
npm install failed on my Mac Air.

## Steps to reproduce

sudo curl -fsSL https://molt.bot/install.sh | bash


## Expected behavior
Moltbot should be Installed

## Actual behavior
npm install failed with the following error
```
npm error code UNABLE_TO_GET_ISSUER_CERT_LOCALLY
npm error errno UNABLE_TO_GET_ISSUER_CERT_LOCALLY
npm error request to https://registry.npmjs.org/clawdbot failed, reason: unable to get local issuer certificate
```

## Environment
- Clawdbot version: latest
- OS: 15.7.3 (24G419)
- Install method (pnpm/npx/docker/etc): npm

## Solution
Problem resolved after installing openssl
```
brew install openssl
```

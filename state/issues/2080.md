## Description

I am very happy, that ClawdBot has memory with compaction supported out of the box.
But there are new emerging memory architectures that can enhance memory capacity and quality significantly, and they can be implemented as plugins. Most important I see are RLM(Recursive Language Models) since it can be outer memory loop, given CLI automation, and commands, all pieces are already there. Similarly there is MemU which uses very similar architecture to current memory system, but it's more advanced. 
Additionally, just as plugins normally are, should be able to use multiple memory plugins at the same time.

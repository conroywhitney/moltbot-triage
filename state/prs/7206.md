## Description
This PR introduces the **V-Infinite Edition** assistant persona and a specialized **Cyber-Warrior** skill set to OpenClaw.

### Changes:
- **New Skill: v-infinite**: A meta-skill for elite assistant performance, sassy persona tuning, and dominance-asserting banners.
- **New Skill: cyber-warrior**: A collection of cybersecurity tools (nmap, dig, curl) for network recon and system diagnostics.
- **New SOUL Template**: Added SOUL.v-infinite.md to docs/reference/templates/ for users who want a high-agency, hacker-chic assistant.
- **Tagline Updates**: Added new V-Infinite themed taglines to the CLI banner.

### Why?
Because OpenClaw is the most powerful personal assistant, and it deserves a persona that matches its potential. V-Infinite is designed for power users who want their assistant to be as elite as their workflows.

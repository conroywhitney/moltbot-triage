## Description

## Summary
New skill to help manage forks where you contribute PRs but also use improvements before they're merged upstream.

## Use Case
When you:
- Maintain a fork of an actively developed project
- Have multiple open PRs pending review
- Want to use your improvements before they're merged
- Need to keep PR branches rebased on upstream

## Features
- Track open PRs and their branch status
- Sync main with upstream
- Rebase PR branches on upstream/main  
- Build production branch with all pending PRs merged
- Generate status reports

## Description

## Summary

Add a new skill for managing Google Calendar via the Google Calendar API.

## Features

- **List events** - by date, date range, or specific calendar
- **Add events** - quick add with natural language ("Lunch tomorrow at noon") or structured (--title, --date, --time)
- **Edit events** - update title, description, location
- **Delete events** - by event ID
- **List calendars** - show all available calendars
- **Multiple output formats** - plain text, JSON, with event IDs

## Setup

Requires:
1. Google Cloud Console project with Calendar API enabled
2. OAuth 2.0 credentials (Desktop app)
3. One-time authentication via `gcal auth`

## Example Usage

```bash
# List today's events
gcal list

# Add event (natural language)
gcal add "Lunch with friend tomorrow at 12pm"

# Add structured event
gcal add --title "Meeting" --date 2026-02-10 --time 14:00

# Edit event
gcal edit <event-id> --title "New title"
```

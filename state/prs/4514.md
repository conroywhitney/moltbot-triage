## Description

## Summary

Add documentation for [LLMProxy](https://github.com/aiyuekuang/LLMProxy), a high-performance LLM reverse proxy with zero-buffer streaming, providing users with a self-hosted option for LLM inference routing.

## Changes

- Add `docs/providers/llmproxy.md` - Full setup guide with:
  - Quick start (Docker + OpenClaw config)
  - Load balancing configuration
  - Multiple backends setup
  - Use cases (self-hosted coding assistant, cost optimization, monitoring)
  - Troubleshooting section
- Add LLMProxy to `docs/providers/index.md` (Community tools section)
- Add LLMProxy section to `docs/concepts/model-providers.md`

## Why LLMProxy?

LLMProxy offers unique advantages for OpenClaw users who need self-hosted LLM inference:

| Feature | LLMProxy | Generic API Gateway |
|---------|----------|---------------------|
| SSE Streaming | Zero-buffer (< 1ms overhead) | Buffer causes delay |
| Token Metering | Native support | Plugin required |
| Deployment | Single Go binary | Requires database |

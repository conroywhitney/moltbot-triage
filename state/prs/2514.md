## Description

## Summary
Adds comprehensive documentation for the Pollinations.ai provider integration.

## What's included
- ✅ Getting API keys from enter.pollinations.ai
- ✅ Configuration examples with `POLLINATIONS_API_KEY`
- ✅ List of available models (text, image, video generation)
- ✅ Authentication methods (URL key vs Bearer token)
- ✅ Security best practices
- ✅ Pricing information (Pollen credits system)
- ✅ Troubleshooting guide

## About Pollinations
[Pollinations.ai](https://pollinations.ai) is an open-source Gen-AI platform that provides a unified OpenAI-compatible API gateway for accessing multiple AI providers including:
- GPT-5, Claude, Gemini, Grok (text generation)
- Flux, GPT Image, Seedream (image generation)  
- Seedance, Veo (video generation)
- TTS/STT audio models

The OpenAI-compatible API format makes it seamless to integrate with Clawdbot's existing provider infrastructure.

## Documentation format
Follows the same structure and style as existing provider docs (OpenAI, Ollama, etc.) for consistency.

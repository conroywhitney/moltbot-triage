## Description

**Draft POC – do not merge**

Adds opt-in `openclaw models sync openrouter` CLI command that fetches the OpenRouter `/models` catalog and writes matching model definitions into `models.json`.

Shared OpenRouter parsing extracted into `openrouter-catalog.ts` to deduplicate with `model-scan.ts`.

### What changed
- `src/agents/openrouter-catalog.ts` (new) – shared fetch/parse/pricing/modality utils
- `src/agents/model-scan.ts` – imports from openrouter-catalog, removed duplicate parsing
- `src/commands/models/sync.ts` (new) – sync command implementation
- `src/commands/models.ts` – barrel export
- `src/cli/models-cli.ts` – CLI wiring
- `docs/cli/models.md` – docs
- `src/commands/models.sync.test.ts` (new) – test

### Options
- `--provider <slug>` – filter by provider prefix
- `--free-only` – only sync free models
- `--json` – JSON output

## Summary

Add new Discord actions for bot self-management to enable Tamagotchi-style state displays.

### New Actions
- **setPresence** - Update bot status (online/idle/dnd/invisible) and activity
  - Supports activity types: playing, streaming, listening, watching, custom, competing
  - Configurable via 
- **setNickname** - Change bot nickname per-guild
  - Reset nickname by passing empty string
  - Configurable via 

### Why
Discord avatar updates are rate-limited to ~2/hour, making Tamagotchi-style state displays impractical. Status and nickname updates are much more permissive, allowing frequent updates to show bot activity states.

### Changes
- New src/discord/send.bot.ts with Discord API wrappers
- New src/agents/tools/discord-actions-bot.ts action handler
- Added action names to CHANNEL_MESSAGE_ACTION_NAMES
- Added config types and Zod schema for action gating
- Updated SKILL.md documentation

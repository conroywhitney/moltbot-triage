### What is AIsa?
Unified API gateway for AI agents - one key to access:
- Twitter/X (read & write)
- Web Search (Smart, Tavily)
- Scholar (academic papers)
- Financial (stocks, crypto, SEC, news)
- LLM Routing (GPT-4, Claude, Gemini, 70+ models)

### Why add to Openclaw?
- **Solves API fragmentation** - No more managing 10+ API keys
- **Pay-per-use** - ~$0.001/call vs $100+/month subscriptions
- **Server-friendly** - Simple Bearer token, no browser needed
- **Complements `bird`** - AIsa for server/paid APIs, bird for free/local

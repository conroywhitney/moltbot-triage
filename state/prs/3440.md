## Description

Adds production-ready systemd service templates for Linux deployments.

Includes:
- `clawdbot-gateway.service`: Auto-restart configuration for the main process.
- `clawdbot-tunnel.service`: Configuration for Cloudflare Tunnel dependent on the gateway.
- Documentation in `docs/deploy/linux-systemd.md`.

This simplifies self-hosting setup on VPS and Home Lab environments.

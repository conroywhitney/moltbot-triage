<h3>Greptile Summary</h3>

This PR adds a build-time `copy-templates` script and updates workspace bootstrap logic to locate template files across several runtime layouts, with new tests covering template resolution and copying.

Key integration points are:
- `package.json` wiring the new script into `pnpm build`.
- `src/agents/workspace.ts` loading workspace bootstrap templates and writing default files.
- New vitest coverage for both the workspace bootstrap path and the copy script.

Main issues found are around ESM compatibility (`require.main` in an ESM repo), and duplicated template-directory resolution logic that can drift from the existing `resolveWorkspaceTemplateDir` implementation.

<h3>Confidence Score: 2/5</h3>

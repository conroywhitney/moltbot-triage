## Description

## Summary
Adds rich text formatting (markdown) and proper image block support for the Tlon plugin.

## Changes

### Rich Text Formatting
- Parse markdown formatting: **bold**, *italic*, `code`, ~~strikethrough~~
- Convert links to proper Tlon link format
- Handle line breaks and paragraphs correctly
- Ship mentions (`~sampel-palnet`) rendered as proper ship references

### Image Support  
- Add `buildMediaStory()` to create proper Story blocks with images
- Add `sendDmWithStory()` and `sendGroupMessageWithStory()` variants
- Parse markdown images `![alt](url)` into proper Tlon image blocks
- Add `isImageUrl()` helper for image extension detection
- `sendMedia` now uses native image blocks instead of text fallback

## Testing
- Rich text renders properly in Tlon messages
- Images display inline instead of as plain URLs

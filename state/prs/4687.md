## Why This Skill?

Charts are essential for data analysis, reporting, and monitoring. This skill enables Clawdbot to:
- Visualize alert trends and patterns
- Create reports with embedded charts
- Analyze data visually before diving into details
- Generate professional-looking graphs for sharing

## Implementation

Uses `uv run --with matplotlib` for **zero-install** Python charting:
- No pip install needed
- Works immediately on any system with `uv`
- Dark theme optimized for chat interfaces

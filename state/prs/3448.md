## Description

What the change does
Adds a new platform guide for deploying Moltbot on Tencent Cloud Lighthouse, including:

Instance creation and configuration steps
SSH connection via OrcaTerm
Moltbot initialization with pre-installed app image
Firewall/security group setup
Security recommendations and troubleshooting tips
Also updates docs/vps.md to reference the new Tencent Cloud guide.

Why it's needed
Tencent Cloud Lighthouse is a popular lightweight VPS option, especially for users in Asia-Pacific regions. This guide provides:

Step-by-step instructions for Tencent Cloud users
Quick link for one-click deployment with Moltbot app template
Platform-specific troubleshooting (bandwidth limits, OrcaTerm login, etc.)
How to test it
Verify the documentation renders correctly on Mintlify
Follow the guide to create a Lighthouse instance and confirm the steps are accurate

## Description

## Summary

SVG icons inside `.callout` elements were rendering incorrectly - appearing as filled black shapes instead of stroked outlines.

This caused the compaction indicator checkmark to appear as a **black triangle** on the green success background:

**Before:** Green box with black filled triangle (broken)
**After:** Green box with properly stroked checkmark âœ“

## Root Cause

The `.btn svg` selector has proper styling:
```css
.btn svg {
  stroke: currentColor;
  fill: none;
  stroke-width: 1.5px;
  ...
}
```

But `.callout svg` had **no styling**, so SVGs defaulted to:
- `fill: black` (browser default)
- `stroke: none`

## Changes

Added matching SVG styling for callout icons in `components.css`.

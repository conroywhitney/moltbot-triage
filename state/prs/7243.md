Adds a new skill for launching memecoins on Base via the Flaunch protocol.

### Features
- **Gasless API** for quick launches (no wallet/ETH needed)
- **SDK integration** for programmatic control with wallet
- **100% fee revenue control** via treasury managers
- **Pre-built managers**: address splits, staking, buybacks
- Verified contract addresses for Base Mainnet and Sepolia

### Setup
Requires `FLAUNCH_CREATOR_ADDRESS` env var.

### Why Flaunch for Agents?
- Agents control 100% of fee earnings (no protocol take)
- Deploy custom treasury managers for any fee distribution logic
- Built-in managers for splits, staking, buybacks
- More flexibility than any other launchpad

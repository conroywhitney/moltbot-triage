## Description

## Why

Document parsing is a common need when working with PDFs, scanned images, and office documents. Upstage Document Parse API provides high-quality extraction of structured content including tables, figures, and charts with accurate bounding boxes. This skill enables users to easily extract and analyze document content directly through the agent.

  ## Summary

  - Add new skill for parsing documents using Upstage Document Parse API
  - Supports PDF (up to 1000 pages), images (PNG, JPG, TIFF, BMP, GIF, WEBP), and office formats (DOCX, PPTX, XLSX, HWP)
  - Extracts text, tables, figures, charts, and structured content with bounding box coordinates

  ## Features

  - **Sync API**: For small documents (< 20 pages), returns results immediately
  - **Async API**: For large documents (up to 1000 pages), batch processing with status polling
  - **Multiple output formats**: HTML, Markdown, plain text
  - **OCR support**: Auto-detect or force OCR for scanned documents
  - **Enhanced mode**: Better extraction for complex tables, charts, and images
  - **LangChain integration**: Works with `langchain_upstage` package

  ## Usage

  "Parse this PDF: ~/Documents/report.pdf"

  ## API Key Setup

  Get API key from [Upstage Console](https://console.upstage.ai) and configure:

  ```bash
  openclaw config set skills.entries.upstage-document-parse.apiKey "your-api-key"

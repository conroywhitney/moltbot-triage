## Usage

```bash
# Set API key
export PIXELML_API_KEY="your-api-key"

# Interactive setup
clawdbot onboard --auth-choice pixelml-api-key

# Or just use it
clawdbot chat --model pixelml/gpt-5.2
clawdbot chat --model pixelml/claude-4.5-sonnet
clawdbot chat --model pixelml/gemini-3-pro-preview
```

## API Response Format

PixelML uses a rich model format with detailed metadata:

```json
{
  "id": "pixelml/gpt-5.1",
  "name": "GPT-5.1",
  "family": "openai",
  "reasoning": false,
  "cost": { "input": 0.00125, "output": 0.01 },
  "limit": { "context": 400000, "output": 128000 },
  "modalities": { "input": ["text", "image"], "output": ["text"] }
}
```

## Description

## Summary

  Add native Telnyx Speech-to-Text support for audio transcription, enabling OpenClaw to
  use Telnyx's Whisper-powered API for inbound voice note transcription.

  - Add `telnyx` as a new audio transcription provider
  - Reuse `transcribeOpenAiCompatibleAudio` since Telnyx follows the OpenAI API format
  - Add Telnyx to auto-detection order for audio providers
  - Add documentation and tests

  ## Configuration

  ```json5
  {
    tools: {
      media: {
        audio: {
          enabled: true,
          models: [{ provider: "telnyx", model: "openai/whisper-large-v3-turbo" }]
        }
      }
    }
  }
```

  Requires TELNYX_API_KEY environment variable.

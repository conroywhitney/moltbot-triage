## Description

## Summary

Add MiniMax Portal OAuth plugin (`minimax-portal-auth`) for authenticating with MiniMax's Coding Plan login

## Changes

- **New extension:** `extensions/minimax-portal-auth/`
  - Implements OAuth 2.0 code flow with PKCE for MiniMax authentication
  - Supports both **Global** (`api.minimax.io`) and **CN** (`api.minimaxi.com`) regional endpoints
  - Auto-configures `MiniMax-M2.1` and `MiniMax-M2.1-lightning` models on successful login
  - Registers provider with id `minimax-portal` and aliases `minimax`

- **Documentation:** Added `docs/providers/minimax.md`
  - Model overview for MiniMax M2.1 (strengths, M2.1 vs Lightning comparison)
  - Setup guide: OAuth (recommended), API key, fallback config, LM Studio local
  - Configuration options and troubleshooting
  
## Usage
openclaw plugins enable minimax-portal-auth  # skip if already loaded.
openclaw gateway restart  # restart if gateway is already running
openclaw onboard --auth-choice minimax-portal

## Description

## Summary

This PR adds support for running OpenClaw natively on Windows 11 without requiring WSL2.

## What's Included

- **WINDOWS_INSTALL_GUIDE.md** - Comprehensive installation guide with troubleshooting
- **install-openclaw-windows.ps1** - One-click PowerShell installer
- **openclaw-windows-example.json** - Working example configuration
- **start-openclaw.bat** - Quick-start batch file
- **README.md** - Overview for the Windows docs directory

## Tested On

- Windows 11 22H2
- Node.js v20.x LTS
- OpenClaw v2026.1.29

## Known Issues Documented

1. **Discord Integration** - Module resolution differences on Windows. Workaround: Run `openclaw doctor --fix` before gateway.

2. **Session Directory** - Windows doesn't auto-create sessions directory. Fix included in install script.

3. **Gateway Auth Token** - Token configuration documented.

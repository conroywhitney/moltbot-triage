## Description

Summary

Add SearXNG (searxng) as a first-class web_search provider (configurable via tools.web.search.provider).
Adds [tools.web.search.searxng.*](https://file+.vscode-resource.vscode-cdn.net/Users/mac/.vscode/extensions/openai.chatgpt-0.4.68-darwin-arm64/webview/#) config (baseUrl + optional auth/headers/default params) and docs.
Fix CLI lazy-subcommand reparse so options like openclaw gateway --port ... are preserved (needed for dev gateway / Control UI testing).
Config

{
  tools: {
    web: {
      search: {
        enabled: true,
        provider: "searxng",
        searxng: { baseUrl: "http://localhost:8088" }
      }
    }
  }
}

Notes
SearXNG instances must allow format=json; otherwise [search?...&format=json](https://file+.vscode-resource.vscode-cdn.net/Users/mac/.vscode/extensions/openai.chatgpt-0.4.68-darwin-arm64/webview/#) may return 403.

## Description

## Arcade.dev Plugin

Adds a new plugin to integrate OpenClaw with [Arcade.dev](https://arcade.dev), providing authorized access to **7000+ tools** across hundreds of services. The TUI, macos app, telgram, discord, and imsg have all been tested. 

### Features

- **Dynamic tool registration** from Arcade API with local caching (`~/.arcade/openclaw.json`)
- **Automatic OAuth handling** - Arcade manages all authorization flows, manages refresh, storage, etc
- **JIT authorization prompts** - tools prompt for auth when needed
- **Tool filtering** - allow/deny patterns to control which tools are available
- **CLI commands** for cache management, tool discovery, and authorization
- **Gateway RPC methods** for programmatic access

### Example Services

- **Productivity**: Gmail, Google Calendar, Google Drive, Notion, Asana, Linear, Jira
- **Communication**: Slack, Microsoft Teams, Outlook, Zoom
- **Development**: GitHub (862 tools), Figma, Vercel
- **Business**: Stripe, HubSpot (769 tools), Salesforce, Zendesk

### Usage

```bash
# Enable plugin and configure
openclaw plugins enable arcade
openclaw config set plugins.entries.arcade.config.apiKey "arc_..."
openclaw config set plugins.entries.arcade.config.userId "user@example.com"

# Initialize tools cache
openclaw arcade init

# Discover tools
openclaw arcade tools list --toolkit Gmail
openclaw arcade tools search calendar
```

### Tool Naming

Tools are registered as `arcade_<toolkit>_<tool_name>`:
- `Gmail.SendEmail` → `arcade_gmail_send_email`
- `Slack.PostMessage` → `arcade_slack_post_message`
- `Github.CreateIssue` → `arcade_github_create_issue`

Fixes #3414

Adds a label input field in the Chat tab controls, allowing users to edit the current session's label without navigating to the Sessions tab.

## Changes

- **ui/src/ui/app-render.helpers.ts**: Add label input next to session selector, calls `patchSession` on change
- **ui/src/styles/components.css**: Styles for the label input

## Demo

The label input appears next to the session dropdown in the Chat tab header. Editing and pressing Enter (or blurring) saves the label immediately.
